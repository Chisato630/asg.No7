# DockerによるMySQLの構築
Dockerインストール後、コンテナ起動、MySQLにログインし、
以下を確認済み。

- movie_list確認
```sh
  mysql> show databases;
  +--------------------+
  | Database           |
  +--------------------+
  | information_schema |
  | movie_list         |
  | mysql              |
  | performance_schema |
  | sys                |
  +--------------------+
  5 rows in set (0.08 sec)
```
- moviesテーブル確認
```sh
mysql> show tables;
+----------------------+
| Tables_in_movie_list |
+----------------------+
| movies               |
+----------------------+
1 row in set (0.00 sec)
``` 
- moviesテーブルのレコード確認
```sh
mysql> select * from movies;
+----+--------------------------------+-----------------------------+
| id | name                           | director                    |
+----+--------------------------------+-----------------------------+
|  1 | ショーシャンクの空に           | フランク・ダラボン          |
|  2 | この世界の片隅に               | 片渕須直                    |
+----+--------------------------------+-----------------------------+
2 rows in set (0.00 sec)
```
